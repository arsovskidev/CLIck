Installation

üöÄ Quick Install (Recommended)

1

Install with pip

# Install CLIck from PyPI
pip install click-tasks

# Or install from source
pip install git+https://github.com/arsovskidev/CLIck.git
2

Add Bash Alias

# Add to your ~/.bashrc or ~/.zshrc
echo 'alias task="click"' >> ~/.bashrc
source ~/.bashrc

# Now you can use either:
click --help
task --help

üîß Development Install

1

Clone Repository

git clone https://github.com/arsovskidev/CLIck.git
cd CLIck
2

Install Dependencies

pip install -r requirements.txt
3

Create Alias

# Add to your ~/.bashrc or ~/.zshrc
echo 'alias click="python $(pwd)/main.py"' >> ~/.bashrc
echo 'alias task="python $(pwd)/main.py"' >> ~/.bashrc
source ~/.bashrc

Note: If you encounter GLIBC compatibility issues with binary releases, use the pip installation method which works on all Python-supported systems.

Quick Start

Add Tasks

# Add a simple task
click a "Fix bug in authentication"

# Add with priority and due date
click a "Review code" --priority high --due tomorrow

# Add with tags
click a "Update docs" --tags "documentation,urgent"

# Using the alias
task a "Deploy to production" --priority high

Manage Tasks

# List all tasks
click l

# List completed tasks
click l --completed

# Complete a task
click c 1

# Delete a task
click d 1

# Using the alias
task l --priority high

Command Reference

Task Management

click a "task" Add new task
click l List tasks
click c <id> Complete task
click d <id> Delete task
task a "task" Add with alias

Bulk Operations

click ca Complete all tasks
click da Delete all tasks
